Height Monitor

  • ECE 153B Project

Project Overview

For our project, we would like to implement a height monitoring device that can take measurements on different objects. The user will press the start button and begin moving the monitor upwards until it reaches the desired height. The accelerometer will be used to detect when the monitor stops moving and the distance between the device and the ground will be obtained from the ultrasonic sensor.
